In the Secure ATM Transactions project, I focused on creating a clear and secure graphical interface. Key considerations included:

  • Security Prominence: Highlighting the biometric verification step.
  • Accessibility: Considering touch target sizes and visual hierarchy.

Fake News Detection Web Platform

For the Fake News Detection project, the goal was to make a complex NLP model accessible to the public through a web interface. The design prioritized ease of use for submitting articles and understanding the classification results. Design choices were made to ensure:

  • Simplicity: A clean input area for text and a clear 'Analyze' button.
  • Transparency: Presenting results (e.g., "Real" or "Fake" with confidence score) in an understandable format.
  • Responsiveness: Ensuring the interface is usable on various screen sizes.
  • Informative Feedback: Guiding users on how to interpret the model's output.
